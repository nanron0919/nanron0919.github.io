Shopback 面試經驗分享

2020 真的是變動很大的一年，長話短說就是，我又要找工作了 😅

休息一段時間後，最近開始比較積極的注意徵才資訊，及跟一些獵人頭聯繫，這次透過躍科的 Viola，介紹到 Shopback，在 onsite interview 之前，會有三關算是遠端面試 (skype)，分別是

人資，大概聊一下之前的經歷

技術主管，大概聊一下之前的經驗

Assignment，需要一星期內 (彈性，可討論) 完成一份作業回傳給他們 (我覺得難度算簡單，有一些 React 經驗應該都會寫)

這裡幫廣告一下，Viola 是我認識的獵人頭中最專業的一位，他會很認真了解雙方的需求，介紹適合你的職缺，不像之前碰到的某些獵人頭，我就已經說不要去銀行了還硬要推 …

在去面試之前，我只知道他們是做現金回饋的服務，只要是透過他們服務到其他電商平台購物，就可以獲得現金回饋

Shopback 的位置在小巨蛋對面，進到辦公室的第一印象是不大，但開放且乾淨。在會議室稍等了一小段時間後，有四名工程師帶著筆電進來，開始介紹今天的面試流程:

  • 第一階段是技術面試，需時約 1.5 小時
  • 第二階段是主管面試，需時約 1.5 小時

技術面試

  • 自我介紹+關於個人經歷的一些問題及討論，包含一些過 往的技術決策
  • 關於矩陣轉換的演算法測試，這邊我用 pseudo code + 口頭解釋，老實講我覺得表現的普普，演算法的確不是我的強項，像 leetcode 的困難項目我向來是跳過的
  • 一些網路基礎，例如 http method、http status code、browser cache 之類的
  • 關於 Redux 的問題 (實作 Connect)，這邊是我覺得最驚艷的部份，我從來沒想過要這樣用 Redux

主管面試

有兩位主管一起進行面試，主要討論了以下幾個問題

  • 前份工作離職的原因
  • 怎麼處理插件
  • 在工作上最不能接受的事情 (會讓你馬上想找下一份工作)
  • 目前 Shopback 的發展及展望
  • Shopback 在徵人上的原則

整體心得

Shopback 在徵人上可以感覺到十分嚴謹，但又不過頭。整體面試過程很累人，但覺得很充實收獲很多。最近這幾年的工作經驗有部份放在建立團隊 (及向上管理) 上，在技術成長方面反而是比較忽略了，經過這次面試看到自己在技術上的不足之處，這是很少在面試中會得到的經驗。

而在跟用人主管的面試上，也可以感受到他們的誠意，是想要真的了解你這個人是不是適合他們，深入且不會用貶抑的方式討論，這樣的面試水準完全命中我對於企業徵才的標準上。這樣的面試印象，即便我無緣加入 Shopback，也會讓我想在未來會幫他們大力宣傳。

不過一位面試主管一直問我有沒有內部認識的人，我印象中應該是沒有? 不知道為什麼要一直問 😂

Updated: 後來才知道，ShopBack 裡面有以前公司，由我面試進來的 RD (不過可惜沒有合作太久的時間)，只能說世界真小，也還好我在工作上有留好名聲給別人探聽

推薦程度: ★★★★★
